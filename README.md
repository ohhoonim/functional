# Java 함수형 프로그래밍

## 개발환경
- JDK: 17
- BUILD: Gradle
- IDE: Visual Studio Code

## 목록
- 람다 표현식 : 람다식 사용법
- 함수형인터페이스 : 필수 인터페이스 4종 예제  
- Stream : map(), filter(), reduce() 
- Collectors : collect()와 Collectors클래스 
- Comparator : sorted() 
- Optional : 결과 체크 
- 스트레티지 패턴 구현 : 파라미커로 람다식 사용하기 
- 데코레이터 패턴 구현 : compose()
- EAM 패턴 구현  : 람다식에서 예외처리 
- Lazy : intermediate, terminal 오퍼레이터 이해하기 
- Map-Reduce : 맵 리듀스 구현 
- Parallel : 병렬화 처리(스레드)
